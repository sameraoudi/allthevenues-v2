# All The Venues Logo Asset Package

Approved logo direction: **Curated Venue Window**.

This package fixes the transparent PNG issue by removing the generated checkerboard background and adding a real alpha channel. The logo artwork itself was not redrawn or regenerated during this fix; the transparent files were post-processed from the approved logo outputs.

## Approved logo palette

- Deep Navy: `#0E1B2A`
- Soft Blue: `#6C93B7`
- Pearl: `#F7F4EF`
- Soft Sand: `#C8B8A0`

## Folders

- `masters/` contains fixed transparent PNGs at the original canvas size and trimmed master versions.
- `web_exports/` contains scaled transparent PNG exports for website usage.
- `favicon_app_icon/` contains app icons and `favicon.ico`.
- `og_social/` contains 1200 x 630 Open Graph/social preview images.
- `previews/` contains a transparency verification preview.

## Recommended usage

- Website header on light background: `web_exports/horizontal_deep_navy/horizontal_deep_navy_1024w.png`
- Website footer on dark background: `web_exports/horizontal_light/horizontal_light_1024w.png`
- Favicon: `favicon_app_icon/favicon.ico`
- Social avatar/app tile: `favicon_app_icon/atv_app_icon_deep_navy_512x512.png`
- Icon-only mark on light background: `web_exports/icon_deep_navy/icon_deep_navy_512x512.png`
- Icon-only mark on dark background: `web_exports/icon_light/icon_light_512x512.png`

Note: These are raster PNG/ICO assets. Editable vector source should be created in a vector design tool before production handoff if precise SVG/PDF vector files are required.
